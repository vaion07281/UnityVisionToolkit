# Installation

## Requirements
* Unity 6000.0 or later.

## via Unity Package Manager (UPM)

1. Open your Unity Project.
2. Go to **Window > Package Manager**.
3. Click the `+` icon in the top left corner.
4. Select **Add package from git URL...**
5. Enter the repository URL (e.g., `https://github.com/vaion07281/UnityVisionToolkit.git`) and click **Add**.

## Local Installation

1. Clone or download the repository to your computer.
2. Open your Unity Project.
3. Go to **Window > Package Manager**.
4. Click the `+` icon in the top left corner.
5. Select **Add package from disk...**
6. Navigate to the downloaded folder and select the `package.json` file.
