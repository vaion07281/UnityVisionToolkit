# Quick Start

Welcome to the Unity Vision Toolkit! Here's how to quickly get started with some of the core features.

## Exploring Samples

The best way to learn is by looking at the examples provided.
1. Open the **Package Manager**.
2. Select **Unity Vision Toolkit** from the list of installed packages.
3. In the package details panel on the right, expand the **Samples** section.
4. Click **Import** next to the sample you want to explore (e.g., EventBus, StateMachine).
5. The samples will be imported into your `Assets/Samples/Unity Vision Toolkit/` folder.
6. Open the included `.unity` scenes (e.g., `EventBusSampleScene.unity`) and press Play in the Editor to see interactive demonstrations of the systems in action.

## Basic Setup

Remember to include the runtime namespace in your scripts:
```csharp
using UnityVisionToolkit.Runtime;
```

Enjoy building your game with the Unity Vision Toolkit!
